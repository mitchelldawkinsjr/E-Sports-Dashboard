# Screenshots Directory

This directory contains screenshots for the User Guide.

## Directory Structure

```
screenshots/
  login/          - Login page screenshots
  navigation/     - Sidebar and navigation screenshots
  organizations/ - Organization management screenshots
  seasons/       - Season management screenshots
  teams/         - Team and roster management screenshots
  matches/       - Match and result submission screenshots
  standings/     - Standings page screenshots
  announcements/ - Announcement screenshots
  notifications/ - Notification screenshots
```

## Adding Screenshots

1. Capture screenshots using the guidelines in `SCREENSHOTS_GUIDE.md`
2. Save them in the appropriate subdirectory
3. Use the exact filenames specified in `SCREENSHOTS_GUIDE.md`
4. Update `USER_GUIDE.md` to reference the screenshots
